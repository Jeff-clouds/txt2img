关于 Markdown 图片生成器

项目介绍

Markdown 图片生成器是一个简单而强大的工具，它可以帮助你将 Markdown 文本转换成精美的图片。无论是分享到社交媒体，还是用于文档展示，都能够保持统一的视觉风格。

主要特性

- 支持完整的 Markdown 语法
- GitHub 风格的暗色主题
- 自定义表情符号装饰
- 自定义水印文本
- 代码高亮显示
- 响应式设计
- 一键生成图片

技术实现

本项目使用纯前端技术实现，主要包括：
- HTML5 用于页面结构
- CSS3 用于样式设计
- JavaScript 用于交互逻辑
- marked.js 用于 Markdown 渲染
- html2canvas 用于图片生成

开源协议

本项目采用 MIT 协议开源，欢迎使用和贡献。
