关于 Markdown 图片生成器

项目介绍

Markdown 图片生成器是一个简单而强大的工具，它可以帮助你将 Markdown 文本转换成精美的图片。提供单图生成和长文分页两种模式，满足不同场景的需求。本工具完全在浏览器中运行，无需后端服务器支持。

主要功能

1. 单图生成模式（主页）：
- 适合短文本和单图展示
- 实时预览渲染效果
- 自定义表情符号装饰
- 自定义水印文本
- 一键生成图片
- 长按保存图片

2. 长文分页模式（子页面）：
- 适合长文本分页展示
- 智能文本分割算法
- 统一固定图片尺寸
- 自动添加水印
- 批量下载支持
- 确保文字完整性

使用说明

单图生成模式：
1. 在主页编辑框中输入短文本
2. 自定义表情符号和水印
3. 点击预览查看效果
4. 生成图片后长按保存

长文分页模式：
1. 点击"长文分页"进入子页面
2. 输入完整的长文本内容
3. 设置表情符号和水印
4. 预览并生成图片
5. 使用"下载全部图片"一键保存

注意事项：
- 分页模式下图片具有统一高度
- 智能分割确保文字不被截断
- 支持代码块、表格、列表等格式
- 每张图片自动添加水印

技术实现

本项目使用纯前端技术实现，主要包括：
- HTML5 用于页面结构
- CSS3 用于样式设计
- JavaScript 用于交互逻辑
- marked.js 用于 Markdown 渲染
- html2canvas 用于图片生成
- Range API 用于智能文本分割
